  • Master Page
    • Add to <head> tag

      Add to <head> tag on Master page.

      <!-- JQuery for animation --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.2.1/jquery.min.js" type="text/css" /> <!-- Google for fonts --> <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Montserrat:400,700" type="text/css" /> <!-- Icons for Chevron --> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.3.0/css/font-awesome.min.css" type="text/css" />
    • Add to <body> tag

      Add jquery script before end of <body> tag on Master page.

      <!-- accordion sections --> <script type="text/javascript"> /*<![CDATA[*/ $('.accordion-toggle').click(function(e) { e.preventDefault(); $("a:first-of-type").removeClass('chev'); var $this = $(this); if ($this.next().hasClass('show')) { $this.next().removeClass('show'); $this.next().slideUp(350); } else { $this.parent().parent().find('li .accordion-inner').removeClass('show'); $this.parent().parent().find('li .accordion-inner').slideUp(350); $this.toggleClass('chev'); $this.next().toggleClass('show'); $this.next().slideToggle(350); } }); /*]]>*/ </script>

  • Target Configuration

    To ensure that Flare recognizes FontAwesome and other custom links in the Master page header, enable Allow Local Stylesheets check box in the project. To enable:

    Project Organizer -> Targets -> [project target file]

    In the target file, in the General tab, check Allow Local Stylesheets

  • CSS

      Add the following code to your CSS.

      .accordion { box-sizing: border-box; font-weight: 300; /* Space section box from left margin*/ padding: 0; /* Hide bullets to the left of section box */ list-style: none; }
  • XML Code

      A single level or base level can contain headings, text and lists.

      <ul class="accordion"> <!-- Wrapper needed for accordion to work --> <li class="accordion-chevron"> <a class="accordion-toggle" href="#"><i class="fa fa-chevron-down"></i>Toggle Section Title</a> <div class="accordion-inner"> <p>Note: Use of <div> tag for -inner.</p> </div> </li> </ul>

      Nested sections are added after each -inner declaration. A nested section consists of:

      <ul class="accordion"> <!-- Wrapper needed for accordion to work --> <li class="accordion-chevron"> <a class="accordion-toggle" href="#"><i class="fa fa-chevron-down"></i>Toggle Section Title</a> <ul class="accordion-inner"> <li class="accordion-chevron"> <a class="accordion-toggle" href="#"><i class="fa fa-chevron-down"></i>Toggle Section Title</a> <div class="accordion-inner"> <p>Note: Parent -inner tag is UL and child -inner tag is DIV</p> </div> </li> </ul> </li> </ul>
